Multi-race stitching

Several races sharing some runners. Each line is one race in the format id:dividend, id:dividend, .... The relative-then-LS calibrator inverts each race independently, removes the per-race translation, and stitches by slope-weighted least squares per item.

How to read it: $\theta$ is the median-centred latent ability shared across races. The bars in the plot show $\theta$ with vertical error bars equal to the per-item standard deviation of the (centred) race-local locations — a quick proxy for stability of the stitch.
